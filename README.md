## Setup Demo
By @jPoechill

To install and start Client App:

  1. Run `cd reactreadable`

  2. Run `npm install`

  3. Run `npm start`

## Setup API Server

To install and start the API Server:

1. Run `cd reactreadableserve`

2. Run `npm install`

3. Run `npm start`
